Tool: Filter FASTA

Tool: checkm2

Tool: Prokka
Version: Galaxy Version 1.14.6+galaxy1
Parameters: Force GenBank Compliance - YES

Step 4) Species ID
Tool: BLAST
Version: BLASTn (web)
Parameters: rRNA/ITS

Step 5) Build phylogenetic tree
Tool: Type Strain Genome Server
Version: https://tygs.dsmz.de/ (web)
Parameters: Whole Proteome Tree

Step 6) Visualize phylogenetic tree
Tool: Interactive Tree of Life
Version: v7 (web)
Parameters: Display bootstraps as text

Step 7) Average nucleotide identity
Tool: ezbiocloud.net/tools/ani
Version: website version
Parameters: Ran two calculations using NCBI genome database starred entries for Salmonella typhimurium and Salmonella bongori. S. typhi. was the closest hit on BLAST, S. bongori was a distant relative. I pulled the FASTA files for each and compared them to my SPAdes filtered contigs from subject DNA to judge relatedness.
